## Installing
```bash
$ git clone https://github.com/HYPER-MOD/multidevice-bot-v3
$ cd multidevice-bot-v3
$ npm start / node .
```

## FOR TERMUX/UBUNTU/SSH USER

```bash
apt update && apt upgrade
apt install git -y
apt install nodejs -y
git clone https://github.com/HYPER-MOD/multidevice-bot-v3
cd multidevice-bot-v3
npm install
```
